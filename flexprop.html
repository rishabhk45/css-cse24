Explanation of Flexbox Properties Used:

  • display: flex; - This property turns the container into a flex container, enabling flexbox layout.
  • flex-direction: row; - This property arranges the flex items in a horizontal row.
  • flex-wrap: wrap; - This property allows the flex items to wrap onto multiple lines if they exceed the container's width.
  • justify-content: space-around; - This property distributes the flex items evenly with space around them.
  • align-items: center; - This property vertically centers the flex items within the container.
  • flex: 1 1 100px; - This shorthand property allows each flex item to grow and shrink as needed, with a base size of 100px.
